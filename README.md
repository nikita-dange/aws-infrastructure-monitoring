# 🚀 DevOps Monitoring Project

## 📌 Overview
This project demonstrates server monitoring using Prometheus, Grafana, and Node Exporter.

## 🧰 Tools
- Prometheus  
- Grafana  
- Node Exporter  
- AWS EC2  

## ⚙️ Features
- CPU, Memory & Disk Monitoring  
- Real-time Dashboard  
- Metrics collection  

## 🚀 Setup
1. Install Prometheus  
2. Install Node Exporter  
3. Configure prometheus.yml  
4. Install Grafana  
5. Connect Prometheus to Grafana
